## 주제 
- Webhook 방식을 사용해 원격 서버에 접속하여 CPU 사용량 등을 측정해 임계값 초과 시 로컬 서버에서 Slack 알림이 오도록 자동화
- Webhook 이란, 웹 서버나 애플리케이션 등에서 특정 이벤트(예: 결제 완료, 주문 생성, 메시지 수신 등)가 발생했을 때, 미리 정해놓은 다른 서비스의 URL(엔드포인트)로 이벤트 정보를 실시간 전송해주는 방식이다.
